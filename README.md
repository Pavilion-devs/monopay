# 7K Mobile Landing Page

React version of the 7K Mobile landing page (converted from the original HTML).

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
```

## Stack

- **Vite** + **React 18**
- **Tailwind CSS** for styling
- **@iconify/react** for icons (Solar icon set)
